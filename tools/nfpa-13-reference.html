What it does

The standard gets renumbered between editions and the section you remember has moved. This is a fast lookup for designers:

  • Convert a section number across NFPA 13 editions
  • Search a roadmap of common rules by topic
  • Skip the index when you just need to find where something lives

How to use it

  1. Enter a section number to see where it maps across editions.
  2. Or search a topic, like small room or obstruction.
  3. Jump straight to the reference.

FAQ

Which NFPA 13 editions does it cover?
It maps common sections across the recent editions designers work between. Always confirm against the adopted edition in your jurisdiction.
Is this the code itself?
No. It is a reference aid to help you navigate NFPA 13 faster. It does not reproduce or replace the standard, which you should consult for the authoritative language.
